// Git Command 

1. git  init
2. git add first.txt (git add likhne ke bad tab press karo or jab vo file a jaye jise push karna hai tan enter press kar do)

Example: -   git add .\first.txt

3. git commit -m "first commit"
4. git branch -M main (ye command hai branch ka nam change karne ke lie)
5. git remote add origin https://github.com/anubhav-singh-dev/solid-meme
6. git push -u origin main
